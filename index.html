Common Event Listeners in HTML

HTML can use events to trigger actions in a browser. These include window event attributes, form events, keyboard events, mouse events, drag events, clipboard events, media events, and more. Here are some of the common events you can have your program "listen" for.


Tag Type of Event What it Does
onerror Window Event Attribute Script to be run when an error occurs
onload Window Event Attribute Runs after a page is finished loading
onpageshow Window Event Attribute Script to be run when user navigates to a page
onchange Form Event Fires when the value of the element is changed
oninput Form Event Script to be run when an element gets user input
onselect Form Event Fires after some text has been selected in an element
onsubmit Form Event Fires when a form is submitted
onkeydown Keyboard Event Fires when a user is pressing a key
onkeypress Keyboard Event Fires when a user presses a key
onkeyup Keyboard Event Fires when a user releases a key
onclick Mouse Event Fires on a mouse click on the element
onwheel Mouse Event Fires when mouse wheel rolls up or down over an element
onmouseover Mouse Event Fires when the mouse pointer moves over an element
ondrag Drag Event Script to be run when an element is dragged
ondrop Drag Event Script to be run when dragged element is being dropped
onscroll Drag Event Script to be run when an element's scrollbar is being scrolled
oncopy Clipboard Event Fires when the user copies the content of an element
oncut Clipboard Event Fires when the user cuts the content of an element
onpaste Clipboard Event Fires when the user pastes some content in an element
